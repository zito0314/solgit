UI 실험 공간

버튼, 카드, 뱃지, 탭 등 SOLGIT에서 쓰일 것 같은 컴포넌트를 마음껏 실험해볼 수 있는 샌드박스입니다.

  • 다크 모드 토글
  • 호버/포커스 상태 디자인
  • 반응형 레이아웃 실험

GitHub Pages 배포

정적 사이트를 GitHub Pages로 배포하는 전체 플로우를 반복해서 익힐 수 있습니다.

  • 코드 작성 → 커밋 → 푸시
  • Settings > Pages 설정
  • 버전 업 시 배포 확인

📊 최근 활동 (Mock 데이터)

실제 Git 연동은 아니고, JS에서 더미 데이터를 뿌려주는 형태입니다.

오늘의 연습 미션

  1. 새 브랜치 feature/ui-upgrade 만들기
  2. 버튼 스타일 조금 수정 후 커밋
  3. GitHub에 푸시하고, 브라우저에서 결과 확인
