# SOLGIT Practice Web

SOLGIT(신한은행 내부 개발 플랫폼)을 공부하기 위한 연습용 웹사이트입니다.  
GitHub 저장소에 웹 파일을 업로드하고, CI/CD 또는 GitHub Pages 배포 연습에 사용할 수 있습니다.

---

## 📦 포함 파일

- `index.html` – 메인 페이지
- `style.css` – 기본 스타일
- `script.js` – 간단한 JS 이벤트
- `assets/` – 로고, 파비콘
- `.gitignore`
